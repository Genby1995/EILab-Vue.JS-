# EILab

## Описание
Приложение на Vue.JS v.3 с роутингом, погрузкой данных и адаптивной версткой на SCSS (тестовое задание для "EILab").

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
